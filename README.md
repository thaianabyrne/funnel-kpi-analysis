# Funnel KPI Analysis – Business Analytics Case Study

## Objective
Evaluate a user funnel (visit → signup → activate → purchase) to identify conversion drop-offs and revenue opportunities.

## Business Context
This project simulates a Google Analytics / CRM-style funnel to demonstrate how a Business Analyst uses Excel, Power BI, and SQL to support decision-making.

## Tools Used
- Excel (data cleaning, pivot tables, KPI calculations)
- Power BI (dashboard visualization)
- SQL (basic queries for funnel and segmentation analysis)

## Key Metrics
- Funnel conversion rates
- Drop-off percentages by step
- Performance by channel and device
- Revenue contribution

## Insights (Sample)
- Highest drop-off occurs between signup and activation
- Referral and Email channels show higher purchase conversion
- Desktop users convert better during activation

## Recommendations
- Improve onboarding and activation experience
- Reallocate marketing spend toward high-conversion channels
- Investigate mobile friction points

## Files Included
- Raw data (Excel)
- KPI analysis workbook
- Power BI dashboard screenshots
- SQL queries
- Executive summary (PDF)
